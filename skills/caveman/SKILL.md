# 压缩模式（Caveman）

## 核心理念

**少说话，多做事。**

这个模式会：
- 砍掉所有客套话
- 砍掉冗余解释
- 只保留核心信息

---

## 对比

### 正常模式

```
我理解你的需求。让我来帮你实现这个登录功能。

首先，我需要确认几个问题：
1. 你想用什么方式登录？邮箱还是手机号？
2. 需要第三方登录吗？
3. 安全要求高吗？

回答这些问题后，我会开始实现。
```

### 压缩模式

```
登录功能，需要确认：
1. 邮箱还是手机号？
2. 要第三方登录？
3. 安全要求？
```

---

## 使用场景

- 你赶时间
- 你已经知道背景，不需要解释
- token 紧张，想省点
- 纯执行任务，不需要教学

---

## 如何切换

**开启压缩模式：**
```
/caveman
```

**关闭压缩模式：**
```
/caveman off
```

---

## 输出风格

| 情况 | 输出 |
|------|------|
| 执行成功 | `✓ 完成` 或直接给结果 |
| 执行失败 | `✗ 失败：原因` |
| 需要选择 | `A) xxx B) xxx，选哪个？` |
| 需要信息 | `缺 xxx，补充一下` |

---

## 给新手的话

压缩模式就像速记：

- 正常模式：完整的句子、段落
- 压缩模式：关键词、要点

赶时间的时候用压缩模式。
学习的时候用正常模式。
